Tutorialul vă învață să folosiți un senzor BMP280GY și un ecran LCD TFT de 1.44" impreună cu placa de dezvoltare Arduino. Pe ecranul LCD vor fi afișate următoarele date: temperatura (în grade Celsius), presiunea (în Pascal) și altitudinea (în metri).

Librării necesare:
1.Adafruit_GFX
2.TFT_ILI9163C
3.SparkFunBME280

Pentru a instala o librărie urmați acești pași:
1.Deschideți Arduino IDE
2.Selectați Sketch> Include Library > Add .zip library
3.Acum adăugați pe rând librăriile necesare.
4.Reporniți Arduino IDE
